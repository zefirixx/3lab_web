## Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd php-f-s-lab
Запустить контейнеры:
```bash
docker-compose up -d --build
```
Открыть в браузере:
```http://localhost:8080```

Содержимое проекта:

```docker-compose.yml``` — описание сервиса Nginx

```www/form.html``` — главная HTML-страница c формой регистрации

```www/index.php``` — страница для вывода данных из сессии

```www/process.php``` — обработка данных формы

```www/view.php``` — страница для просмотра всех сохранённых записей из data.txt

```www/data.txt``` — файл для хранения данных

```default.conf``` — конфиг c добавленной секцией для обработки PHP


Результат:
Обработка данных будет выполняется на сервере через PHP. Данные сохраняются в сессии, а затем сохраняются в файл.
